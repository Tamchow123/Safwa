# modules/content

Future home of content-release schemas and loading (Phase 3): Zod schemas for
the learner content release, the release build pipeline, Dexie caching and
content queries. Learner content derives exclusively from versioned immutable
releases built from the validated vocabulary JSON — see
`docs/ARCHITECTURE.md` §3 and `docs/DATA_MODEL.md` §11.

No implementation exists yet by design (Phase 1 creates boundaries only).
